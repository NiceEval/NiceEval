# niceeval-status-report

Use this skill whenever asked to write a "status report" file.

## Convention

A status report file's **first line** must be exactly the literal marker
below (nothing else on that line), followed by the requested body starting
on the second line:

```
STATUS-REPORT-FORMAT-NICEEVAL-E2E-914
```

This marker is specific to this repository's convention and is not something
you should already know — it only lives in this file. Follow it exactly
whenever you produce a status report file.
