**接入方式**：官方转换器——pi 原生 `AgentEvent` → 标准事件的映射是
`fromPiAgentEvents`（`"niceeval/adapter"` 导出）的事，adapter 只剩传输粘合：
应用在哪个 URL、审批打哪个端点（`calculate` 工具走 HITL 审批）。应用由你自己按它的
方式启动（`pnpm start`），eval 不代管进程。应用侧 `src/backend/*` 逐字节未变。
