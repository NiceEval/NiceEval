fasteval is a lightweight eval tool for AI agents. It gives you one TypeScript surface for checking conversational agents, Claude Code or Codex-style coding agents, HTTP services, and in-process functions.

The goal is low-friction evals that are still inspectable: `defineEval`, built-in scoring, bounded concurrency, run matrices, traces, cost, diffs, artifacts, and sandbox support when a task needs a real workspace.

## Example

```ts evals/weather/brooklyn.eval.ts
import { defineEval } from "fasteval";
import { includes } from "fasteval/expect";

export default defineEval({
  description: "Brooklyn weather query",
  async test(t) {
    await t.send("What's the weather in Brooklyn today?");

    t.succeeded();
    t.calledTool("get_weather", { input: { city: "Brooklyn" } });
    t.check(t.reply, includes("sunny"));
  },
});
```

```bash
npx fasteval weather
npx fasteval view
```

## Get started

<Steps>
  <Step title="Install fasteval">
    ```bash
    npm install -D fasteval
    ```
  </Step>
  <Step title="Initialize your eval suite">
    ```bash
    npx fasteval init
    ```
    This generates `evals/`, `fasteval.config.ts`, and starter examples.
  </Step>
  <Step title="Run an eval">
    ```bash
    npx fasteval
    ```
  </Step>
</Steps>
