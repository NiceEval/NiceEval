NiceEval is an agent-native eval tool for AI agents. It gives you one API surface for checking conversational agents, Claude Code or Codex-style coding agents, and custom agent workflows.

The goal is low-friction evals that are still inspectable: `defineEval`, built-in scoring, bounded concurrency, run matrices, traces, cost, diffs, artifacts, and sandbox support when a task needs a real workspace.

## Example

```ts evals/weather/brooklyn.eval.ts
import { defineEval } from "niceeval";
import { includes } from "niceeval/expect";

export default defineEval({
  description: "Brooklyn weather query",
  async test(t) {
    await t.send("What's the weather in Brooklyn today?");

    t.succeeded();
    t.calledTool("get_weather", { input: { city: "Brooklyn" } });
    t.check(t.reply, includes("sunny"));
  },
});
```

```bash
npx niceeval exp local weather
npx niceeval view
```

## Get started

<Steps>
  <Step title="Install niceeval">
    ```bash
    npm install -D niceeval
    ```
  </Step>
  <Step title="Initialize your eval suite">
    ```bash
    npx niceeval init
    ```
    This generates `evals/`, `niceeval.config.ts`, and starter examples.
  </Step>
  <Step title="Run an eval">
    ```bash
    npx niceeval exp local
    ```
  </Step>
</Steps>
